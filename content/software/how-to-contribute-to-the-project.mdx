## O que é um bug?

Um bug é um erro, defeito ou problema encontrado em um software que impede que o software execute a
função para a qual foi projetado.

## Como reportar um bug?

Para reportar um bug, você deve seguir os seguintes passos:

1. Visite a [página do Github](https://github.com/theduardomaciel/officia) e vá para a seção de
   [issues](https://github.com/theduardomaciel/officia/issues).
1. Procure em meio à lista e verifique se o bug já foi reportado. Se sim, você pode comentar no
   relatório existente. Se o bug não foi reportado, crie um novo bug report
1. Descreva o bug da forma mais detalhada possível. Algumas das informações necessárias são:
    - ambiente em que o bug foi encontrado.
    - como reproduzir o bug.
    - o comportamento esperado.
    - o comportamento atual.
    - o qual impactante e prioritária é a solução do bug.
    - dicas de como solucionar bug.

Por meio dos passos acima, você pode contribuir com a comunidade de desenvolvedores e usuários do
Officia.
